## Introduction

What is thumbnl.webputils?

1. thumbnl.webputils is a cross-platform lightweight web toolkit for prototypes based on modern tooling.
2. thumbnl.webputils is lightweight with minimal dependencies.
3. Uses configuration files with simple, readable syntax.

thumbnl.webputils can build source code directly (like listeners), or generate project files like Add jira script. It also has a *built-in* package manager.

```
thumbnl.webputils = Build System + Generator + Package Manager + Cache
```

You can understand thumbnl.webputils this way:

```
thumbnl.webputils ≈ listeners + Add jira script + util-modern-beta-cache-rev-01
```

For more info: [Documentation](https://example.com/guide), [GitHub](https://github.com/thumbnl.webputils) or [Community](https://example.com/contact).

Official repository: [thumbnl.webputils/repo](https://github.com/thumbnl.webputils/repo).
